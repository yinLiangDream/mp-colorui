## 特色

- 全部采用 TypeScript 编写，类型声明一目了然
- 丰富的配色方案，极具色彩表现力
- 丰富的组件库，应有尽有

## 使用案例

### 引入 `CSS` 文件

引入 `CSS` 需要在 `src` 目录下 `app.scss` 中引入样式

```js
@import "~mp-colorui/dist/style/index.scss";
```

### 引入组件

```jsx
import { ClAvatar } from "mp-colorui";

<ClAvatar shadow shape="round" size="large" />;
```
