## 参数说明

### Accordion 参数

| 参数      | 说明               | 类型    | 可选值 | 默认值    |
| --------- | ------------------ | ------- | ------ | --------- |
| title     | _手风琴标题_       | string  | -      | -         |
| open      | _是否展开_         | boolean | -      | _`false`_ |
| animation | _开启动画效果_     | boolean | -      | _`true`_  |
| card      | _是否使用卡片形式_ | boolean | -      | _`false`_ |
| speed     | _展开速度_         | number  | -      | _`0.15`_  |

### Accordion 事件

| 事件名称 | 说明           | 返回参数 |
| -------- | -------------- | -------- |
| onClick  | 点击手风琴事件 | boolean  |
