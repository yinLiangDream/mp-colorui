## 使用说明

```jsx
import {ClScreenDrawer} from 'mp-colorui'
```



## 一般用法

### 显示全屏抽屉

```jsx
<ClScreenDrawer show >我是内容</ClScreenDrawer>
```



## 参数说明

### ScreenDrawer 参数

| 参数         | 说明                | 类型    | 可选值             | 默认值    |
| ------------ | ------------------- | ------- | ------------------ | --------- |
| show         | *显示 screenDrawer* | boolean | *`true`*,*`false`* | *`false`* |
| renderPage   | 页面显示内容        | Element | -                  | -         |
| renderDrawer | 抽屉显示的内容      | Element | -                  | -         |



### ScreenDrawer 事件

| 事件名称 | 说明                 | 参数返回 |
| -------- | -------------------- | -------- |
| onHide   | *点击返回小箭头触发* | -        |
